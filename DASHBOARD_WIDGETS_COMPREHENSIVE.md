## ðŸ”— **DATA SOURCE MAPPING**

Each widget will pull from specific API endpoints:

- **Financial Data**: `/api/financial/:jobId/*`
- **Job Data**: `/api/jobs/:id/*`
- **Task Data**: `/api/tasks/*`
- **Time Entry Data**: `/api/timeEntries/*`
- **Material Data**: `/api/materialRequests/*`, `/api/purchaseOrders/*`
- **Inventory Data**: `/api/inventory/*`
- **Progress Reports**: `/api/financial/:jobId/progress-reports`
- **SOV Data**: `/api/jobs/:id/sov-components`
- **Work Orders**: `/api/workOrders/*`
